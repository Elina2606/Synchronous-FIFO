RTL design and verification of Synchronous FIFO using Verilog
# Synchronous FIFO using Verilog

## Overview
Designed a synchronous FIFO for buffered data transfer in digital systems.

## Features
- Parameterized FIFO depth
- Full and Empty flag generation
- Read/Write operations
- Testbench verification

## Tools Used
- Verilog HDL
- Vivado / ModelSim
- GTKWave

## Applications
- Data buffering
- SoC communication
- FPGA designs
